# Qu Content

教材制作で参照するMarkdown、原資料メモ、下書きを置く共有ディレクトリです。

この配下のMarkdownは `scripts/embed-content.ts` の対象になります。資格別ディレクトリ名から `certification_slug` が推定されます。

## ディレクトリ

| Path | Slug | 用途 |
|---|---:|---|
| `it-passport/` | `ip` | ITパスポート |
| `basic-information-engineer/` | `fe` | 基本情報技術者 |
| `generative-ai-passport/` | `genai` | 生成AIパスポート |
| `g-test/` | `gken` | G検定 |
| `spi/` | `spi` | SPI |
| `boki/` | `boki` | 簿記 |

## 推奨構成

- `references/`: 参照文献、URLメモ、出典整理
- `source-materials/`: 教材化前の原資料メモ
- `drafts/`: レッスン本文や問題の下書き

著作権のある本文を丸ごと保存せず、引用は最小限にして、出典URL・書誌情報・要約を残します。
