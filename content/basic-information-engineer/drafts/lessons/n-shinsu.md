# N進数の概念

## このレッスンでできるようになること

- コンピューターが2進数を使う理由を説明できる
- 2進数、10進数、16進数の違いを説明できる
- ビットとバイトの関係を説明できる
- 「同じ数を別の表記で表しているだけ」という感覚を持てる

## なぜコンピューターは2進数を使うのか

コンピューターは、内部では電気信号を使って処理を行います。電気信号は大きく分けると「オン」と「オフ」の2つの状態で扱えます。

この2つの状態を数字に置き換えると、オンを `1`、オフを `0` と表せます。つまり、コンピューターは `0` と `1` の組み合わせで情報を表現しています。

1つの `0` または `1` を表す最小単位を **ビット** と呼びます。8ビットをまとめた単位が **バイト** です。

```text
1ビット = 0 または 1 のどちらか
8ビット = 1バイト
```

## ビットを増やすと表現できる数が増える

1ビットで表せる状態は、`0` と `1` の2通りです。

2ビットになると、次の4通りを表せます。

```text
00
01
10
11
```

3ビットになると、8通りを表せます。

```text
000 001 010 011
100 101 110 111
```

ビット数が1つ増えるたびに、表現できるパターン数は2倍になります。

```text
1ビット: 2通り
2ビット: 4通り
3ビット: 8通り
4ビット: 16通り
```

このように、`0` と `1` だけでも大量に組み合わせれば、数値、文字、画像、音声など多くの情報を表現できます。

## N進数とは

**N進数**とは、N種類の記号を使って数を表す方法です。

私たちが普段使っている10進数は、`0` から `9` までの10種類の記号を使います。10個集まると桁が1つ上がります。

一方、2進数は `0` と `1` の2種類の記号だけを使います。2個集まると桁が1つ上がります。

16進数は、`0` から `9` と `A` から `F` の16種類の記号を使います。

| 表記 | 使う記号 | 桁が上がるタイミング |
|---|---|---|
| 2進数 | `0`, `1` | 2個集まったとき |
| 10進数 | `0` から `9` | 10個集まったとき |
| 16進数 | `0` から `9`, `A` から `F` | 16個集まったとき |

## 同じ数を別の方法で表しているだけ

たとえば、10進数の `17` は、2進数では `10001`、16進数では `11` と表せます。

```text
10進数: 17
2進数 : 10001
16進数: 11
```

表記は違いますが、表している数は同じです。

これは「1時間」「60分」「3600秒」が同じ時間を表しているのと似ています。単位や表し方が違うだけで、指している量は同じです。

## 文字も0と1で表される

コンピューターは文字も `0` と `1` の組み合わせで扱います。

文字にはそれぞれ番号が割り当てられており、その番号を **文字コード** と呼びます。代表的な文字コードには `Shift_JIS` や `Unicode` があります。

同じ「あ」という文字でも、文字コードの種類によって割り当てられる番号は異なります。作成時と表示時で異なる文字コードを使うと、文字化けが起こることがあります。

## 要点

- コンピューターは電気信号のオン・オフを `1` と `0` として扱う
- `0` または `1` の1つ分をビットと呼ぶ
- 8ビットを1バイトと呼ぶ
- N進数は、N種類の記号で数を表す方法
- 2進数、10進数、16進数は、同じ数を別の表記で表しているだけ
